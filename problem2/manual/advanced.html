Problem 2 Manual

The Hack: Create a website that people can visit to find out about movie offerings in Halifax.

Step 0.0 From Todo to Done

Before going over the actual code used in the website it is useful to note the resources used in the making of this hack.

  • jQuery
  • Twitter Bootstrap

Step 0.1 Finding the Starting Point

Navigate to the /Hack_A_Week/problem2/ directory in your Windows Explorer / File Manager.

Step 0.2 What is Done

Once inside the problem's directory, find the /done/ directory. You should see a list of files:

  • index.html
  • style.css
  • main.js

Step 0.3 Copy & Paste

Decide where you want to store your project, then copy the files in the done directory found from above into your new private directory. Your final result should look like this:

/index.html
/css/style.css
/js/main.js

Congrats! You now have a working full solution and code to play with! If you want to learn more about the sample solution's code please continue reading below and feel free to ask any question that this manual does not answer for you :).
